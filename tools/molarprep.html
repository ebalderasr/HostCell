MolarPrep

Preparación de reactivos y buffers sin errores. Convierte entre unidades de concentración y calcula masas y volúmenes al instante.

Qué es
Una calculadora de soluciones molares para el laboratorio bioquímico
MolarPrep es una PWA enfocada en las conversiones más comunes al preparar reactivos de laboratorio: de molaridad a gramos, de stock concentrado a solución de trabajo, y entre unidades de concentración (M, mM, µM, g/L, %, w/v).

Incluye una base de datos de compuestos comunes en bioprocesos para autocompletar el peso molecular, acelerando el flujo de trabajo en bancada.
Qué hace
Calcula masas, volúmenes y concentraciones para preparar cualquier solución
MolarPrep cubre los escenarios más frecuentes en laboratorio:
  • ¿Cuántos gramos pesar para preparar X mL de solución Y molar?
  • ¿Cuánto volumen de stock tomar para obtener la concentración deseada?
  • Conversión entre M, mM, µM, g/L y porcentaje (w/v)
  • Preparación de buffers con múltiples componentes
Por qué es importante
La correcta preparación de reactivos es la base de todo experimento reproducible
Un error en la concentración de un buffer o reactivo propaga incertidumbre a todas las mediciones del experimento. Errores de unidades (confundir mM con µM, o g/L con %) son comunes cuando se hacen los cálculos de forma manual o con una calculadora genérica.

MolarPrep reduce este riesgo al ofrecer una interfaz específica para química de soluciones, con validación de unidades integrada.
Cómo lo hace
Fórmulas fundamentales de química de soluciones con base de datos integrada
MolarPrep implementa las ecuaciones fundamentales: n = m/MM, C = n/V y C₁V₁ = C₂V₂. La base de datos de compuestos incluye pesos moleculares de reactivos comunes (glucosa, glutamina, NaCl, HEPES, PBS, entre otros) con autocompletado al escribir el nombre.

La interfaz detecta qué variable el usuario quiere calcular y despeja automáticamente la ecuación correcta, minimizando la posibilidad de error.
