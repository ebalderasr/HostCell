YSI 2950 Live App

Upload a BioSample file. Review replicates. Export cleaned results.

This GitHub Pages app processes raw BioSample*.csv files entirely in the browser. No install, no Python environment, no server upload.

Replicates are grouped by PlateSequenceName, BatchName, WellId, and ChemistryId.
Bench Workflow
  1. Drop one or more YSI BioSample*.csv files
  2. Inspect mean, SD, CV, and flagged replicates
  3. Export summary, outliers, annotated measurements, and manifest
