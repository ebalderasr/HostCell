How are outliers detected?

Three independent statistical tests run on each replicate group (≥ 3 replicates required). A replicate is recommended for discard when it accumulates enough flags (configurable via Consensus signals).

Modified Z-Score
Uses the median and MAD (median absolute deviation) — robust to non-normal distributions. Flags a replicate if |0.6745 × (value − median) / MAD| exceeds the threshold.
IQR Fence
Flags values that fall outside Q1 − multiplier × IQR or Q3 + multiplier × IQR. Requires a non-zero IQR.
Leave-One-Out CV
Tests whether removing each replicate reduces the group CV below the threshold. Only the replicate whose removal gives the greatest improvement is a candidate.

The cleaned mean/SD is computed after applying the recommended discard. Groups are marked REVIEW if no single outlier is identified but CV still exceeds the threshold.
